텍스트 태그와 css

                    < b > 글씨 굵게 < /b >
                    < strong > 글씨 굵게 (문장 강조) < /strong >
                    < h1~6 > 글의 제목 < /h1 >
                    < p > 문단의 표현 < /p >
                    < br > 텍스트 줄바꿈 < /br >
                    < blockquote > 텍스트의 인용문 (안쪽 들여쓰기) < /blockquote >
                    < q >  텍스트의 인용문 (줄바꿈 없음) < /q >
                    < pre > 텍스트에 서식 지정 (작성한 내용 그대로 표현) < /pre >
                    < figure > 독립적인 컨텐츠 표현 < /figure >
                    < hr > 장면 전환, 구획 내 주제 변경, 문단 분리 < >
                    < i > 이탤릭체 < /i >
                    < em > 텍스트의 강조, 중첩 사용 가능 < /em >
                    < mark > 텍스트 중요 표시 < /mark >
                    < small > 덧붙이는 글, 저작권과 법률 같은 거 < /small >
                    < sup > 위로 글씨 첨자 < /sup >
                    < sub > 아래로 글씨 첨자 < /sub >
                    < del > 제거 텍스트 < /del > 
                    < insert > 추가된 텍스트 < /insert >
                    < u > 텍스트에 밑줄 < /u >
                    < kdb > < /kdb > Ctrl+R
                
                font-size: 20px; > 글씨 크기를 20px로 설정
                font-weight: 700; > 글씨 굵기를 700으로 설정, 100 단위, 클수록 굵어짐
                font-family:'궁서'; > 글꼴 설정
                color: red; > 글씨 색상
                text-decoration: underline; > 글씨 장식 설정
                text-transform: lowercase; > 대문자를 소문자로 변환
                text-align: center; > 텍스트 정렬 (수평 정렬)
                letter-spacing: -2px; > 글자 사이의 간격
                line-height: 30px' > 행 높이

공간(영역) 태그와 css

                    < table >
                        < tr >
                            < td > &lg /td <
                            < td > &lg /td <
                        < /tr >
                        < tr >
                            < td > &lg /td <
                            < td > &lg /td <
                        < /tr >
                    &lg /table <    
                    > 테이블 태그, 줄과 행 표현, 줄 tr, 행 td 

                    < div > < /div > > 한 블럭을 표현, 지정한 크기만큼 영역 표시 가능

                    < fieldset > < /fieldset > > 그룹 지정

                    < header > < /header > > 메뉴, 로고, 머리 부분

                    < main > < /main > > 본문 내용

                    < footer > < /footer > > 바닥글, 페이지 정보 등

                    < nav  > < /nav > > 메뉴 사용

                    < section > < /section > > 본문의 글 주제별로 사용

                    < span > < /span > > div보다 작은 영역 (줄바꿈 없음)
                
                    display > 공간 차지 속성 (block, inline-block, flex, grid 등)
                    position > 태그의 위치 지정 (top, right, left, bottom)
                    width > 태그의 너비
                    height > 태그의 높이
                    margin > 태그의 바깥쪽 여백
                    padding > 태그의 안쪽 여백
                    background > 태그의 배경색, 이미지 지정
                    border > 태그의 테두리선
                    box-shadow > 그림자 효과
